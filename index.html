Koneksi Supabase

Hubungkan ke Supabase agar data bisa diakses dari HP maupun komputer. Lakukan langkah berikut satu kali saja:

1
Buka supabase.com → Sign up gratis → klik New Project. Isi nama project (misal: farclo), pilih region Southeast Asia, buat password DB.
2
Di project Anda → buka menu SQL Editor → klik New Query → paste seluruh isi file farclo_schema.sql yang disertakan → klik Run.
3
Buka Project Settings → API. Salin Project URL dan anon public key, lalu paste di bawah.

Kredensial disimpan di localStorage browser ini saja. Data bisnis tersimpan di Supabase milik Anda.
